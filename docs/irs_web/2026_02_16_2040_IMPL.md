1. Start backend: `cd setup && docker-compose up -d`
2. Start frontend: `cd microservices/irs_web && pnpm dev`
3. Open: `http://localhost:3000`

## ⚠️ Risks & Mitigations

| Risk               | Impact         | Mitigation                             |
| ------------------ | -------------- | -------------------------------------- |
| API slow response  | Poor UX        | Debounce + loading states              |
| Large bundle size  | Slow load      | Code splitting, lazy loading           |
| 3D performance     | Jank           | Lazy load, simple scene                |
| CORS issues        | API calls fail | Backend CORS config                    |
| Mobile performance | Slow on mobile | Optimize animations, reduce complexity |
